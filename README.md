# Java CLI Calculator

A simple command-line calculator built with Java.

## Operations
* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Modulo (%)
* Power (^)

## Features
* Interactive command-line interface.
* Input parsing for numbers and operators.
* Error handling for division by zero.
* Includes JUnit 5 tests to verify math logic.

## How to Run
1. Compile the source: `javac src/JavaCalc.java`
2. Run the application: `java src/JavaCalc`
3. To run tests: Use your IDE's JUnit runner on `test/JavaCalcTest.java`.
